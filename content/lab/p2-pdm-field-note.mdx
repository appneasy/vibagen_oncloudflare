## AppSheet PdM app ที่สร้างเอง → Dashboard ที่แจกให้ใครก็ได้ฟรี

โรงงานแห่งหนึ่งมีเครื่องจักร 36 ตัว ใน 8 สถานี ทีม maintenance ต้องตรวจสอบ vibration, temperature, current draw ตามรอบ 1–30 วัน

ทำ AppSheet app ขึ้นมาจัดการ — ใช้งานได้ดีมาก ทีมภาคสนามชอบ

**ปัญหา:** ผู้จัดการอยากเห็น dashboard overview แบบ real-time แต่ไม่มี AppSheet license และไม่รู้จะ share view ยังไงให้คนข้างนอกดูได้โดยไม่ต้อง login

## Data Model: ENIHour_PdM

ระบบมี 10 tables, 49 parameters, 36 machines

```text
Equipment ─has many──► Measurement ─has many──► Measurementdetail
    │                       │                        └──► ParameterID ──► Parameter
    │                       └──► GroupID ─────────────────────────────► Groupphase
    │
    └──has many──► Machine ─has many──► Meter_log      // hour meter บันทึก
                       └──has many──► Part_usage ──► Part_master

Telegramsetup   // standalone · alert config
```

Virtual Columns (AppSheet expressions):

| Column | Expression |
|---|---|
| `Lastest_10_measurementdetail` | SELECT top 10 by date |
| `Latest_part_usage` | MAXROW by timestamp |
| `Part_usage_over_lastest` | SUM usage since last service |
| `Part_master_active` | FILTER where Active = TRUE |

## 4 Insights จาก Technical Extract

:::insight{number=1 title="gviz CSV API — ไม่ต้อง auth ถ้า sheet เป็น public"}
Google Sheets มี endpoint ที่ fetch ได้โดยตรงจาก JavaScript:

`https://docs.google.com/spreadsheets/d/[SHEET_ID]/gviz/tq?tqx=out:csv&sheet=SHEET_NAME`

ไม่ต้อง API key ไม่ต้อง OAuth — แค่ Sheet เป็น public ก็ดึงได้ทันที
:::

:::insight{number=2 title="ข้อมูล real-time ทุกครั้งที่ refresh"}
HTML dashboard fetch CSV ใหม่ทุก load

ทีมภาคสนามบันทึกใน AppSheet → ผู้จัดการ refresh browser → เห็นข้อมูลล่าสุดทันที

ไม่ต้องมี webhook ไม่ต้องมี polling service
:::

:::insight{number=3 title="ไฟล์เดียว แจกได้ทันที"}
HTML file ขนาด ~15KB ส่งผ่าน Line/Email ได้เลย

เปิดใน Chrome/Edge โดยไม่ต้องติดตั้งอะไร ทำงานบน PC, tablet, mobile

ไม่มี hosting ไม่มี domain ไม่มี maintenance
:::

:::insight{number=4 title="ข้อมูลจริง: 36 machines, 11 equipment"}
DigesMC, ScrewPress, เครื่องกระเทาะ, Purify, Separator, Engine GAS — ครบทุก platform ของโรงงาน

ดึงมาแสดงโดยตรงจาก Sheets ที่ทีมภาคสนามอัพเดทอยู่แล้วทุกวัน
:::

:::lesson{title="AppSheet สำหรับ field capture → vibecoded HTML สำหรับ management"}
AppSheet เก่งเรื่อง **เก็บข้อมูลภาคสนาม** — ทีมสนามรัก form และ offline sync

แต่ถ้าอยากให้ management ดู overview ได้ทุกที่ทุกเวลา vibecoded HTML dashboard ที่ดึง Google Sheets โดยตรง **เร็วกว่า ถูกกว่า และ share ได้กว่า**

ไม่ต้องเลือก — ใช้ทั้งสองอย่างตามจุดแข็งของแต่ละเครื่องมือ
:::
